BK7238 Matter方案
==============================



概述
------------------------------
Matter 是一个统一的开源应用层连接标准，旨在使开发人员和设备制造商能够连接并构建可靠、安全的生态系统，并提高联网家庭设备之间的兼容性。
它采用经过市场验证的互联网协议 (IP) 技术构建，并与 Thread 和 Wi-Fi 网络传输兼容。

BK7238 已支持适配matter 1.1，用户只需下载支持matter 版本 SDK，即可进行matter 应用开发。
